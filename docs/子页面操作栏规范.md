# 子菜单页面 · 操作栏按钮规范

从「仓库管理」起，所有子菜单内容页中 **新增 / 修改 / 删除 / 导出** 四个操作按钮统一为 **描边 + 浅色底** 的语义色样式（与参考图一致）。

## 引用样式

在页面 `<head>` 中（建议在 Font Awesome 之后）增加：

```html
<link rel="stylesheet" href="css/subpage-toolbar.css" />
```

路径请按实际 HTML 与 `css` 文件夹的相对位置调整。

## HTML 示例

```html
<div class="wh-toolbar">
  <button type="button" class="btn-toolbar btn-toolbar--primary">
    <i class="fa-solid fa-plus" aria-hidden="true"></i> 新增
  </button>
  <button type="button" class="btn-toolbar btn-toolbar--success">
    <i class="fa-solid fa-pen-to-square" aria-hidden="true"></i> 修改
  </button>
  <button type="button" class="btn-toolbar btn-toolbar--danger">
    <i class="fa-solid fa-trash-can" aria-hidden="true"></i> 删除
  </button>
  <button type="button" class="btn-toolbar btn-toolbar--warning">
    <i class="fa-solid fa-download" aria-hidden="true"></i> 导出
  </button>
</div>
```

新页面若无 `.wh-toolbar` 样式，可使用同文件内的 **`.subpage-toolbar`** 作为操作栏容器（白底 + 上下分割线），与 `warehouse.html` 中工具栏效果一致。

## 语义类名

| 类名 | 用途 | 色系 |
|------|------|------|
| `btn-toolbar btn-toolbar--primary` | 新增 | 蓝 |
| `btn-toolbar btn-toolbar--success` | 修改 | 绿 |
| `btn-toolbar btn-toolbar--danger` | 删除 | 红 |
| `btn-toolbar btn-toolbar--warning` | 导出 | 橙 |
